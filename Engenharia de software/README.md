Atividade 1
Engemharia de software apesar do nome, não se trara de algo tangível assim como as outras engenharias (civil,aeronáutica,mecânica),
no ramo de software não conseguimos enxergar o que foi projetado assim como um prédio, um avião ou um automóvel.

Atividade 2
Engenharia de software não é só projetar um aplicativo mas um processo de longo prazo para mante-lo valioso e sustenável ao longo do tempo
para que continue atual e relevante, como não temos ainda estas respostas nos baseamos na experiência da Google para reagirmos as mudamças 
e tornar um programa susrtentável ao longo da tempo.
